# Cloud artifacts for Lab 10

This directory contains the files prepared for the cloud deployment part of Lab 10.

- `huggingface/Dockerfile` is the Docker Space image definition
- `huggingface/README.md` is the Space README with the required YAML frontmatter
- `teardown.md` documents how to clean up the Space and the optional tunnel

The Hugging Face Space is a separate Git repository, so these files should be copied into that Space repository before pushing to Hugging Face.
