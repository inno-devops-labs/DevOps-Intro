# Teardown

## Hugging Face Space

1. Go to https://huggingface.co/spaces/fleter/quicknotes
2. Click Settings → Delete this Space → confirm

## ghcr.io package

1. Go to https://github.com/fleter?tab=packages
2. Select `quicknotes` → Package settings → Delete this package → confirm

## Cloudflare Tunnel (Bonus)

Stop the `cloudflared` process — the ephemeral tunnel URL expires automatically.
No account cleanup needed for quick tunnels.
