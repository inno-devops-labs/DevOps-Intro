 	Task 1 — SSH Commit Signature Verification



 	Зачем нужны подписанные коммиты

Подписанные коммиты позволяют понять, кто именно сделал изменения в коде.

Также подпись гарантирует, что коммит не был изменён после его создания.



 	Что я сделал.

 SSH-ключ добавлен в GitHub как signing key

 Git настроен для подписи коммитов через SSH

 Коммит создан с подписью

 На GitHub у коммита отображается статус Verified



 	Почему подпись коммитов важна в DevOps

В DevOps часто вносятся изменения в код, инфраструктуру и т.д.

Подпись коммитов помогает доверять этим изменениям, защищает от подмены кода

и упрощает контроль и аудит изменений в проекте.
