# IP 10.91.48.***.60132 > 10.90.137.30.53: 12157+ A? nl.globaltoys.site. (36)

# Анализ:
# Зафиксирован DNS-запрос типа A к домену github.com.
# Трафик передаётся по UDP через порт 53.
# После запроса получен корректный DNS-ответ от сервера.
# Это подтверждает успешное разрешение доменного имени
