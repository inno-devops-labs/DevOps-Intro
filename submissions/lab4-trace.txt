QuickNotes — POST /notes packet/HTTP trace
===========================================

HOW THIS FILE WAS PRODUCED (run on YOUR machine; tcpdump needs root):

  # Terminal 1 — start the app
  cd app/ && go run .

  # Terminal 2 — capture loopback traffic on :8080 into a pcap
  sudo tcpdump -i lo -n -s 0 -w lab4.pcap 'tcp port 8080' &

  # Terminal 3 — generate one request
  curl -v -X POST http://localhost:8080/notes \
       -H 'Content-Type: application/json' \
       -d '{"title":"lab4","body":"trace me"}'

  # stop the capture, then decode it to this text file
  sudo pkill tcpdump
  tcpdump -r lab4.pcap -n -tttt -S > lab4-trace.txt
  # (-S = absolute TCP seq numbers, easier to annotate the handshake)
